# Titon

A premium analytics platform designed for structural clarity and insightful navigation.

## Project info

This project provides a modern, dark-themed landing page with advanced animations and a focus on premium aesthetics.

## How can I edit this code?

You can edit this code using your preferred IDE.

1.  **Clone the repository**
2.  **Install dependencies**
    ```sh
    npm install
    ```
3.  **Start the development server**
    ```sh
    npm run dev
    ```

## Technology Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- GSAP
- Lenis (Smooth Scrolling)

## Features

- **Futuristic 2D Background**: Animated mesh gradients and film-grain textures.
- **Conceptual Visuals**: Abstract guidance markers and illumination effects.
- **Premium Typography**: Bold, minimalist industrial aesthetic.
- **Smooth Momentum Scrolling**: Physics-based navigation.
